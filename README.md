# Design → Code Playbook
Если ты пришёл сюда после одной из моих статей про Figma → Code, то ты по адресу — здесь лежат реальные шаблоны, которые можно сразу забрать в работу

**Статьи:**
* Гайд для малышей: как перегнать дизайн из Figma в код — [Habr](https://habr.com/ru/articles/1018064/) 𐄂 [VC](https://vc.ru/design/2843404-gaid-po-perenosu-dizaina-iz-figma-v-kod) 𐄂 [dsgners](https://dsgners.ru/makstorch/40389-gayd-dlya-malyishey-kak-peregnat-dizayn-iz-figma-v-kod) 𐄂 [Medium](https://medium.com/@makstorch/guide-how-convect-figma-design-to-code-de91265b87e7)<sup>RU</sup> 𐄂 [Medium](https://medium.com/@makstorch/guide-how-to-turn-figma-designs-into-code-2e1bc5d5c2b8)<sup>EN</sup>
* _Гайд для подростков — когда-нибудь_

Моя цель — собрать повторяемый workflow, который помогает: переносить дизайн из Figma в код, уменьшать хаос при работе с AI, ускорять вёрстку, сохранять качество и внимание к деталям. Короче, меньше вайб-кодинга ради вайба, больше рабочих процессов, которые реально доводят до релиза
